# calc24-muti
一个基于UDP协议的多人算24点游戏
----------------

#Feature

 - 支持房间分组机制，支持多人，多房间对战
 - GoRouting并发，但是我也不知道性能咋样

#Usage

    go get github.com/xuzhenglun/calc24-muti
    go build service.go
    go build client.go
